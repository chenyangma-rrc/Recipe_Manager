# Recipe Manager

A full-stack recipe manager built with Next.js, React, PostgreSQL, Prisma, and Auth.js.

## Features

- Public recipe browsing
- Category pages
- User registration and login
- Protected dashboard
- Authenticated recipe creation

## Tech Stack

- Next.js App Router
- React
- TypeScript
- PostgreSQL
- Prisma
- Auth.js
